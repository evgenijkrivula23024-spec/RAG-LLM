# PDF to Text Converter for RAG Pipeline

Скрипт для извлечения текста из PDF-файлов и сохранения его в формате, пригодном для использования в RAG (Retrieval-Augmented Generation) пайплайне.

## 📋 Описание

Данный скрипт предназначен для предобработки документов перед их использованием в системе RAG. Он извлекает текстовое содержимое из PDF-файлов, объединяет их в один текстовый документ и сохраняет в формате UTF-8.

## 🎯 Назначение

- Извлечение текста из текстовых PDF (не отсканированных)
- Объединение нескольких PDF в один текстовый файл
- Подготовка данных для последующей нарезки на чанки и векторизации
- Создание входного файла для RAG-пайплайна

## ⚙️ Требования

### Системные требования
- Python 3.8+
- 1 GB RAM (минимально)
- 100 MB свободного дискового пространства
